FOLIUM LIBRARY NOTES

dir(folium)
help(folium.Map)  <---press Q to exit the help

Can use the .Map() method to generate a base map according to the dimensions of the parameters

map=folium.Map(location[80,-100])   <--creates the map object, however you still need to convert to HTML/JavaScript. 
                                        We do this by saving as an html object:

                                            map.save("Map1.html")

Features of the map using help(folium.Map) are called children of the map.
To add points to the map:
    map.add_child(folium.Marker())
    map.save("Map1.html")

Create a feature group for adding children (features) to your map:
    fg=folium.FeatureGroup(name="NBA Map")
    fg.add_child(folium.Marker(location[38.2,-99.1],popup="I am a marker",icon=folium.Icon(color='green')))
    map.add_child(fg)
